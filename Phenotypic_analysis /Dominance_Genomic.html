1. Introduction

Genomic prediction involves building a model on previously available information to predict future outcomes easily.

In this study, we will optimize the predictive accuracy of genomic models by including dominance and additive effects in GBLUP and machine learning method. The dominance effects will be added through locus-specific weights on the heterozygous SNPs markers before calculating the genomic relationship matrix. This method will be compared to the classical method of using the additive Genomic relationship matrix and additive plus dominance genomic relationship matrix. Furthermore, finally combining the locus-specific weighted Dominance effect genomic data with the Gradient boosting framework.

In summary, four models will be compared;

  1. Genomic BLUP model with additive effects
  2. Genomic BLUP model, including additive and dominance effects
  3. Genomic BLUP model using locus-specific weighted dominance effect genomic data for the genomic relationship matrix.
  4. Machine learning model using locus-specific weighted dominance effect genomic data (Gradient boosting framework).

The analysis was done with G2F data(2018 - 2021), using the yield phenotype

### a function that finds outliers using the resiuals from a linear model and remove the outliers.

outlier.resid<- function(data, trait){ 
  
  data<- data[complete.cases(data[,paste(trait)]),]
  
  #Fit a linear model for the residuals
  Model  <- lm(paste0(trait, "~  Hybrid_Env + Replicate"), data=data)
  
  ##Residual
  data$RESID <- resid(Model)
  
  #finding out what two standard deviations is and save it 
  
  SD2 <- 2 * sd(resid(Model))
  
  # Flag the outliers as 1
  out_flag<-ifelse(data$RESID > SD2, 1, 0) 
  
  # Combine the orginal data
  out_data<-cbind(data, out_flag) # Combine the orginal data
  
  write.table(out_data,file = 'Pheno_data_Outlier.csv',col.names = T,row.names = F,sep = '\t',quote = F)
  
  #Make a new data.frame with no outliers
  out_data<- out_data[!out_data$out_flag,]
  
  write.table(out_data,file = 'Pheno_data_without_Outlier.csv',col.names = T,row.names = F,sep = '\t',quote = F)
  
  return(out_data)
}

### Mixed model 
mixed <- function(fix = NULL, y, kk) {
  n <- length(y)
  y <- as.matrix(y)
  if (is.null(fix)) {
    fix <- matrix(1, n, 1)
  } else {
    fix <- as.matrix(fix)
  }
  ##Negative nloglikelihood Function
  g <- length(kk)
  nloglik_REML <- function(pm) {
    v_phi <- 0
    for (p in 1:g) {
      v_phi <- v_phi + kk[[p]] * pm[p]
    }
    v_sigma <- diag(n) * pm[g + 1]
    v <- v_phi + v_sigma + diag(1e-09, n)
    v_i <- solve(v, tol = -50)
    beta <- solve(t(fix) %*% v_i %*% fix, t(fix) %*% v_i %*% y)
    nloglik <- 0.5 * (unlist(determinant(v))[[1]] + unlist(determinant(t(fix) %*%
                                                                         v_i %*% fix))[[1]] + t(y - fix %*% beta) %*% v_i %*% (y - fix %*% beta))
    return(nloglik)
  }
  parm0 <- rep(1, g + 1)
  parm <- optim(par = parm0, fn = nloglik_REML, method = "L-BFGS-B", hessian = FALSE,
                lower = 0)
  v_phi <- 0
  for (p in 1:g) {
    v_phi <- v_phi + kk[[p]] * parm$par[p]
  }
  v_sigma <- diag(n) * parm$par[g + 1]
  v <- v_phi + v_sigma
  v_i <- solve(v, tol = -50)
  beta <- solve(t(fix) %*% v_i %*% fix, t(fix) %*% v_i %*% y)
  res <- list(v_i = v_i, var = parm$par[-(g + 1)], ve = parm$par[g + 1], beta = beta)
  return(res)
}

mixed <- function(fix = NULL, y, kk) {
  n <- length(y)
  y <- as.matrix(y)
  if (is.null(fix)) {
    fix <- matrix(1, n, 1)
  } else {
    fix <- as.matrix(fix)
  }
  ##Negative nloglikelihood Function
  g <- length(kk)
  nloglik_REML <- function(pm) {
    v_phi <- 0
    for (p in 1:g) {
      v_phi <- v_phi + kk[[p]] * pm[p]
    }
    v_sigma <- diag(n) * pm[g + 1]
    v <- v_phi + v_sigma + diag(1e-09, n)
    v_i <- solve(v, tol = -50)
    beta <- solve(t(fix) %*% v_i %*% fix, t(fix) %*% v_i %*% y)
    nloglik <- 0.5 * (unlist(determinant(v))[[1]] + unlist(determinant(t(fix) %*%
                                                                         v_i %*% fix))[[1]] + t(y - fix %*% beta) %*% v_i %*% (y - fix %*% beta))
    return(nloglik)
  }
  parm0 <- rep(1, g + 1)
  parm <- optim(par = parm0, fn = nloglik_REML, method = "L-BFGS-B", hessian = FALSE,
                lower = 0)
  v_phi <- 0
  for (p in 1:g) {
    v_phi <- v_phi + kk[[p]] * parm$par[p]
  }
  v_sigma <- diag(n) * parm$par[g + 1]
  v <- v_phi + v_sigma
  v_i <- solve(v, tol = -50)
  beta <- solve(t(fix) %*% v_i %*% fix, t(fix) %*% v_i %*% y)
  res <- list(v_i = v_i, var = parm$par[-(g + 1)], ve = parm$par[g + 1], beta = beta)
  return(res)
}

5. Genomic BLUP model using locus specific weighted dominance effect genomic data for the genomic relationship matrix.

This model combines additive and dominance genetic effects by creating a combined additive and dominance genetic relationship matrix. The genetic relationship matrix was constructed using modified SNPs markers where locus-specific weighted dominance effects have been captured in the heterozygous genetic position.

\[ y = 1_u + Z_gG + e \] The above formula is the same as the additive effect model, except that G replaces A, and it is a combination of both additive and dominance effects.

To modify the heterozygous SNPs markers, assuming the degree of dominance differs at different genetic positions. We first estimate the dominance degree based on the heterozygous and homozygous marker genotypes. The degree of dominance is calculated with the below equation;

\[ X_{min} = min(X_{homo1},X_{homo2}) \] \[ X_{max} = max(X_{homo1},X_{homo2}) \] \[ D= 2*(X_{heter}-X_{min}) / (X_{max}-X_{min}) \] Furthermore, use this dominance degree calculated above as weights on the heterozygous genotype to construct a new SNP marker matrix. The boundaries for transformation are; \[ 0, if D 0 \\ D, if 0 < D < 0\\ 2, if D 2 \]
